データのバックアップ(GitHub連携)

手持ちの選択状態はブラウザ(localStorage)に自動保存されますが、閲覧データの削除や端末変更で消えることがあります。GitHubのアクセストークン(このリポジトリ限定のFine-grained PAT)を設定すると、手持ち・ピン留め・除外の状態をGitHubに保存し、別端末からも読み込めます。トークンはこの端末のブラウザ内にのみ保存されます。
トークンの作り方: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token。 Repository accessで「Only select repositories」→ palworld-breeding-chart を選択し、 PermissionsのContentsをRead and writeにして発行した「github_pat_」で始まるトークンを貼り付けてください。
